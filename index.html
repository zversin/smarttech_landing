Кейсы

Примеры IT-задач, где мы сохранили инфраструктуру и добавили новые возможности

КазУТБ: корпуса 1-3

Модернизация без замены: сохранили Perco и добавили Face ID с аналитикой по расписанию.

Что сделали:

  • Сохранили Perco KT-02.9 и Perco Web, доработали турникеты под Face ID.
  • Установили по 2 Face ID на каждый турникет и синхронизировали базу.
  • Настроили отчеты по посещаемости преподавателей и студентов.

Результат:

  • Увеличили парк турникетов с 7 до 10 без замены системы.
  • Единая отчетность по посещаемости в разрезе расписания.

🏃 Спортивный комплекс

Новый объект: запустили доступ и перенесли фото из Perco Web в Hik Central.

Что сделали:

  • Развернули контур доступа в спортивном комплексе.
  • Перенесли фото и базу пользователей из Perco Web в Hik Central.
  • Настроили права доступа и учет посещений.

Результат:

  • Запуск без повторного сбора фото и данных.
  • Готовый контур контроля доступа с первого дня.

🏠 Общежитие

Новый объект с миграцией данных и интеграцией с Z5R Web для учета в МФЦ.

Что сделали:

  • Развернули новый контур доступа в общежитии.
  • Автоматически перенесли фото и данные пользователей.
  • Интегрировали турникеты с Z5R Web для учета в МФЦ.

Результат:

  • Учет и отчеты доступны в МФЦ без ручного ввода.
  • База пользователей перенесена без потери данных.

О компании SmartTech

Мы делаем IT-проекты в Астане и по всему Казахстану: интеграции, автоматизация, аналитика и развитие существующих систем. Начинаем с аудита и предлагаем решения, которые не требуют выбрасывать то, что уже работает.

Опыт в системах контроля доступа - лишь часть нашей экспертизы. Мы связываем оборудование и ПО, переносим данные и выстраиваем отчетность для руководства.

Работаем официально по договору: фиксируем требования, согласовываем архитектуру, запускаем пилот и сопровождаем решение после внедрения.
